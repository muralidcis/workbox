Just to show how to create a rest api service using Eclipse, Maven, jax-ws with jersy over resin 4 webserver

Create a dynamic web project make sure web.xml is generated along with it

Configure the dynamic project to maven project, so the dependency for lib files can be added without any issues

POM.XML

^dependencies> ^dependency> ^groupId>junit^/groupId> ^artifactId>junit^/artifactId> ^version>4.8.2^/version> ^scope>test^/scope> ^/dependency> ^dependency> ^groupId>com.sun.jersey^/groupId> ^artifactId>jersey-server^/artifactId> ^version>1.19^/version> ^/dependency> ^dependency> ^groupId>com.sun.jersey^/groupId> ^artifactId>jersey-client^/artifactId> ^version>1.19^/version> ^/dependency> ^dependency> ^groupId>com.sun.jersey^/groupId> ^artifactId>jersey-core^/artifactId> ^version>1.19^/version> ^/dependency> ^dependency> ^groupId>com.sun.jersey^/groupId> ^artifactId>jersey-json^/artifactId> ^version>1.19^/version> ^/dependency> ^dependency> ^groupId>com.sun.jersey^/groupId> ^artifactId>jersey-servlet^/artifactId> ^version>1.19^/version> ^/dependency> ^dependency> ^groupId>com.sun.jersey^/groupId> ^artifactId>jersey-bundle^/artifactId> ^version>1.19^/version> ^/dependency> ^/dependencies>

Added the servlet class, package init and path in the web.xml files

Example: Jersey Rest service com.sun.jersey.spi.container.servlet.ServletContainer com.sun.jersey.configure.property.packages com.formsdirect.rest.example 1 Jersey Rest service /rest/*

create a package
